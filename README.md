# JEE Math Solver

Simple step-by-step JEE problem solver using OpenRouter.

- Enter problem text
- Get multiple approaches
- Expand steps for details

## Deploy to Vercel

1. Push this repo to GitHub.
2. Import project on Vercel.
3. No build or devDependencies needed for deployment (Vercel auto-detects Vite).
4. Deploy.

Users enter their own OpenRouter API key in the app.
